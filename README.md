# WEBTOONS Library System - CRUD PERN-Stack Web-App

This app allows users to add, leave ratings and reviews, and delete WEBTOONS from their library. 

* User account system with JWT Authentication
* Web-scraping from WEBTOONs website for titles and authors
* Interactive Search Bar
* Rating and Review Features for each WEBTOON
